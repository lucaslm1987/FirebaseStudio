# **App Name**: GCMobile

## Core Features:

- Create Occurrence Report: Create digital occurrence reports.
- Consult Occurrence Report: Consult and search for existing occurrence reports stored in the database.
- Create Daily Service Report: Create daily service reports including details of the service performed.
- Consult Daily Service Report: Consult and search daily service reports saved in the database.
- Create Summons: Create digital summons records.
- Consult Summons: Consult summons records stored in the database.
- Unique Code Authentication: Implement a system to send a unique access code to a registered device

## Style Guidelines:

- Primary color: Deep blue (#003049) to convey trust and security, reflecting the serious nature of law enforcement.
- Background color: Light blue (#D4E0E9), a desaturated shade of the primary, for a calm and professional feel.
- Accent color: Warm orange (#D62828), analogous to the primary, to highlight critical actions and alerts, such as report submission errors.
- Body and headline font: 'Inter' (sans-serif) for clear and accessible readability on tablets, maintaining a modern and efficient aesthetic.
- Use clear, professional icons from a set like Material Design Icons to ensure instant recognition and ease of use on a tablet interface.
- Design the layout to be responsive and tablet-optimized, with larger touch targets and adaptable grids for efficient data entry and review.
- Incorporate subtle transitions and feedback animations to confirm actions, enhancing the user experience without distracting from the task at hand.
